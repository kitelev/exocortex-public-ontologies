## Файловая структура
Каждая папка - это онтология.

Каждый файл в папке это репрезентация RDF-ресурса внутри онтологии. Название файла - это идентификатор ресурса внутри онтологии.

Исключение составляет только файл, с одноименным папке названием с префиксом `!` - это репрезентация RDF-ресурса онтологии

## Содержимое файлов
В каждом файле сначала размещается frontmatter-блок, который является репрезентацией всех RDF-property ресурса. Название frontmatter-проперти полностью соответствует названию RDF-property

Исключение составляет только проперти `!`, которое используется только в ресурсах класса [[owl/owl__Ontology]] для обозначения URL онтологии, так как это единственное, для чего нет проперти в онтологиях rdf, rdfs и owl

Порядок проперти желательно держать такой, как и в RDF-онтологии

Вместо алиаса `a` используется полноценный `rdf:type`

Значением проперти `rdfs:isDefinedBy` указывается wikilink на [[owl/owl__Ontology]]
